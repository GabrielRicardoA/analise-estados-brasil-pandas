# Análise de Dados dos Estados Brasileiros com Pandas

Mini projeto de análise de dados utilizando Python e Pandas com dados públicos da Wikipédia.

## Objetivos do Projeto

- Ler tabelas HTML com Pandas
- Limpar e transformar dados
- Realizar filtros e análises
- Criar classificações populacionais e econômicas
- Praticar análise exploratória de dados (EDA)

## Tecnologias Utilizadas

- Python
- Pandas
- Jupyter Notebook

## Fontes dos Dados

### População dos Estados
https://pt.wikipedia.org/wiki/Lista_de_unidades_federativas_do_Brasil_por_população

### PIB dos Estados
https://pt.wikipedia.org/wiki/Lista_de_unidades_federativas_do_Brasil_por_PIB

## Análises Realizadas

### População
- Top 10 estados mais populosos
- Estados acima de 10 milhões de habitantes
- Estados do Nordeste
- Estado mais populoso
- Estado menos populoso
- Classificação populacional

### PIB
- Estados com PIB acima de 500 bilhões
- Top 10 maiores PIBs
- Top 10 menores PIBs
- Classificação econômica
